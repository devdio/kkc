## Move

### move forward
```python
from roboid import *

albert = AlbertAi()

# move_forward()
albert.move_forward()
wait(300)

# move_forward(cm)
albert.move_forward(10)
wait(3000)

# move_forward(cm, velocity 100%)
albert.move_forward(10, 100)
```
